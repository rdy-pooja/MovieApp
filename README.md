# MovieApp
A REST API for a movie application like IMDB, Using ASP.Net Core Web API to build the API.

GET: api/Movies
GET: api/Movies/5
GET - To Get all the movies.

POST: api/Movies
Post - To Add a movie.

PUT: api/Movies/5
Put - To Update the movie details.

DELETE: api/Movies/5
Delete - To Delete a movie entry.
